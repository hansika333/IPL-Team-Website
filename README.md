❤️ Favourite IPL Team - Royal Challengers Bengaluru (RCB)

About the Project

This project is a fan website dedicated to Royal Challengers Bengaluru (RCB), one of the most popular teams in the Indian Premier League (IPL).

The website provides information about RCB, including team players, statistics, achievements, news updates, and a fan interaction section. It is designed using HTML, CSS, and JavaScript with a modern and attractive user interface.

Features

- 🏏 About Royal Challengers Bengaluru
- 👥 RCB Squad and Player Profiles
- 📊 Team Statistics
- 📰 Latest RCB News
- ❤️ Fan Zone Contact Form
- 📱 Responsive Design
- ✨ Modern UI with Glassmorphism Effects

About RCB

Royal Challengers Bengaluru (RCB) is a professional cricket franchise based in Bengaluru, Karnataka. Established in 2008, RCB has built one of the largest and most passionate fan bases in the IPL. The team has featured legendary players such as Virat Kohli, AB de Villiers, Chris Gayle, and many more.

RCB won its maiden IPL title in 2025, creating a historic moment for millions of fans around the world.
